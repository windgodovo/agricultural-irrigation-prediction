# Agricultural Irrigation Prediction System

**Self-taught Machine Learning Project for AI/ML Job Applications**

## 📋 项目概述
本项目开发了一个农业灌溉需求预测系统，通过分析土壤、天气和作物相关数据，预测农田的灌溉需求量，帮助实现智能农业和节水灌溉。

这是一个端到端的机器学习项目，涵盖数据探索、特征工程、模型训练与可视化，展示了将AI应用于实际农业场景的能力。

## 🛠️ 技术栈
- **数据处理**：Pandas, NumPy
- **可视化**：Matplotlib, Seaborn
- **机器学习/深度学习**：TensorFlow / Keras / PyTorch
- **AI辅助开发**：Gemini （用于代码生成、调试和优化）
- **环境**：Google Colab / Jupyter Notebook

## 📊 主要工作与成果
- 使用Pandas和NumPy对农业数据集进行清洗、缺失值处理和特征工程
- 通过Matplotlib和Seaborn可视化土壤湿度、温度、降雨量与灌溉需求的关系
- 构建预测模型（回归或分类），评估模型性能 Macro f1-score
- 利用Gemini加速模型调参和代码优化，显著提升开发效率
- 生成可行动的灌溉建议，支持智能农业决策

## 📥 数据来源
- 数据集来自Kaggle（Irrigation Water Requirement Prediction Dataset 或类似农业灌溉数据集）
- **注意**：原始压缩包文件较大，未包含在仓库中
- 下载链接：[Kaggle数据集页面](kagglehub.competition_download('playground-series-s6e4'))  

运行前请先下载数据集并解压到 `data/` 文件夹，或使用notebook中的自动下载代码。

## 🚀 如何运行
1. 打开 `agricultural_irrigation_prediction.ipynb`
2. 在Google Colab中运行所有单元格
3. 如需自动下载数据，请运行notebook开头的下载与解压代码

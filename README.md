# Agricultural Irrigation Prediction System

**Self-taught Machine Learning Project for AI/ML Job Applications**

## 📋 Project Overview
Developed an end-to-end machine learning system to predict agricultural irrigation needs based on soil moisture, weather conditions, and crop data. The goal is to support smart farming and water conservation through data-driven decisions.

This project demonstrates a complete ML pipeline: data exploration, feature engineering, model training, evaluation, and visualization.

## ✨ Highlights
- Built a predictive model achieving **Macro F1-score** on irrigation demand
- Performed comprehensive EDA and feature engineering with Pandas & NumPy
- Visualized key relationships between weather/soil factors and irrigation needs
- Leveraged Gemini for faster code generation, debugging, and hyperparameter tuning

## 🛠️ Tech Stack
- **Data Processing**: Pandas, NumPy
- **Visualization**: Matplotlib, Seaborn
- **Machine Learning**: TensorFlow / Keras / PyTorch
- **AI Assistance**: Gemini (code generation & optimization)
- **Environment**: Google Colab / Jupyter Notebook

## 📊 Key Work & Results
- Cleaned and preprocessed agricultural dataset (handling missing values and feature engineering)
- Created insightful visualizations showing correlations between soil moisture, temperature, rainfall, and irrigation requirements
- Trained and evaluated prediction models with strong performance metrics
- Generated actionable irrigation recommendations to optimize water usage

## 📥 Dataset
- Source: Kaggle Irrigation/Water Requirement Dataset
- **Note**: The original compressed dataset is large and **not included** in this repository (to keep it lightweight)
- Download: [Kaggle Dataset](kagglehub.competition_download('playground-series-s6e4'))  ←
- Run the automatic download & extraction code in the notebook if available

## 🚀 How to Run
1. Open `机器学习_灌溉预测.ipynb` (or the main notebook)
2. Run all cells in Google Colab
3. Download the dataset and place it in a `data/` folder (or use auto-download code)
